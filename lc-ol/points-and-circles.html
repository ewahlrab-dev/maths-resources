The Circle
Centre(0, 0)
Radiusr = 3
x² + y² = 9
The Key Rule

To determine where a point lies relative to a circle, calculate its distance from the centre and compare it to the radius.

> Distance from centre greater than radius OUTSIDE
= Distance from centre equal to radius ON
< Distance from centre less than radius INSIDE
Formula:
d = √(x² + y²)
(for a circle centred at the origin)
Worked Examples  — circle: x² + y² = 9  (r = 3)
Point A
(−3, 4)
d = √(x² + y²) d = √((−3)² + 4²) d = √(9 + 16) d = √25 d = 5
5 > 3
OUTSIDE
Point B
(1, 1)
d = √(x² + y²) d = √(1² + 1²) d = √(1 + 1) d = √2 d ≈ 1.41
√2 < 3
INSIDE
Point C
(−3, 0)
d = √(x² + y²) d = √((−3)² + 0²) d = √(9 + 0) d = √9 d = 3
3 = 3
ON
