Leaving Certificate · Ordinary Level Statistics

Mean & Standard Deviation
on the Casio ClassWiz

A step-by-step guide to using the Statistics mode on your calculator.

Example 600, 470, 170, 430, 300
1
Go to the Home screen
Press Home
Press the Home button. You will see the main menu showing five options: Calculate, Statistics, Table, Ratio and Math Box.
What you'll see
2
Select Statistics
Arrow keys → highlight Statistics → OK
Use the arrow keys to move the highlight to Statistics, then press OK.
Statistics selected
3
Choose 1-Variable
Select 1-Variable → OK
The screen shows two options: 1-Variable and 2-Variable. Select 1-Variable and press OK.

Use 1-Variable whenever you have a single list of data values.
Choose 1-Variable
4
The data entry table appears
Empty x and Freq columns — ready to fill in
A table appears with two columns: x (your values) and Freq (frequency). The Freq column fills with 1 automatically for each value — leave it as is.
Empty table
5
Enter your data
Type each number → press EXE after each one
Type each value and press EXE after each one. The screen scrolls as you go — don't worry if earlier values scroll off.

Run the calculation

Your results
What to ignore: The results screen also shows Σx, Σx², σ²x, and s²x — you do not need these for most Leaving Cert questions. Focus only on x̄ for the mean and σx for the standard deviation.
